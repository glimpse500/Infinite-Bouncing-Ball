# Introduction
A simple game simulates a ball moving in the screen.
The ball bounces when it hit the wall or a barrier.
The Bounce Times is recorded and plus one each time the ball bounce

# Installation
Your need to insatall Python 2.7 (https://www.python.org/download/releases/2.7/) and pygame (http://www.pygame.org/download.shtml)  before playing the game.

#How to operation:

1. Use the left button of your mouse to add a barrier on the screen
2. Use the right button of your mouse to remove a barrier on the screen
